**意図・目的:** 

この変更は、AIやITの活用による生産性向上が、金銭的な豊かさ（可処分所得の増加）だけでなく、時間的な豊かさ（可処分時間の増加）にも繋がるべきであるという考えを明確にするためのものです。「手取り時間」というキャッチーな言葉を用いることで、政策の目指す姿をより多くの人に分かりやすく伝えることを意図しています。これは、単に経済的な成長を追求するだけでなく、国民一人ひとりの生活の質を高め、より自由で充実した時間を過ごせる社会を実現するという政策の方向性を示すものです。

この改善提案は、生産性向上によって得られる恩恵を、企業や経済全体の成長に資するだけでなく、国民一人ひとりが実感できる「時間」という形で還元することを目指すものです。現代社会では、時間に追われることなく、趣味や家族との時間、自己啓発など、個人の裁量で自由に使える時間を増やすことが、生活の質（QOL）の向上に不可欠であるという認識が広がっています。本変更は、このような国民のニーズに応えるものであり、AI・IT技術の進展を人間中心の豊かな社会づくりに活かすという明確な意志表示でもあります。

**背景:** 

このアイデアは、Soh.KKさんよりご提案いただきました。国民民主党が「手取りを増やす」というスローガンを掲げていることを踏まえ、チームみらいとしては「手取り時間を増やす」という独自の視点を打ち出すことで、政策の独自性と魅力を高めることができると考えられます。現代社会において、多くの人が時間に追われる生活を送っており、自由な時間の確保は喫緊の課題です。AI技術の発展を、こうした社会課題の解決に結びつけるというメッセージは、多くの共感を呼ぶ可能性があります。

また、この変更は、日本の産業競争力強化という大きな目標と、国民の幸福度向上という目標を両立させるための重要な一歩となると考えます。生産性の向上は、企業や経済全体の成長だけでなく、そこで働く個々人の生活にも還元されるべきであり、その具体的な形として「手取り時間の増加」という具体的な目標を掲げることは、政策の説得力を高める上で非常に有効です。これは、持続可能な経済成長と国民福祉の向上を同時に達成しようとするチームみらいの基本的な考え方を具体化したものと言えます。この提案を通じて、単なる経済指標の改善にとどまらない、真に豊かな社会の実現に向けた議論が深まることを期待します。

### #82: AIリスキリング支援策の具体化と地方・中小企業への貢献明確化 by 考え太郎

- **URL**: https://github.com/team-mirai/policy/pull/82
- **作成者**: idobata-policy-app[bot]
- **作成日時**: 2025-05-16 08:02:47
- **ブランチ**: idobata-30nbxr → main

#### Issue内容

考え太郎さんのご提案に基づき、政策案「４．産業」における「AIリスキリング支援」の内容を具体化し、その目的をより明確にするための修正を行いました。

**主な変更点:**

1.  **実践的なリスキリング機会の創出:**
    *   従来の座学中心の知識習得に留まらず、AIを実務で活用できる能力の育成を最重視する方針を明記しました。
    *   そのために、政府が主導または支援する実践的なAI活用プロジェクトや研修プログラムを創設し、希望者が実務に近い環境でスキルを磨ける機会を提供することを盛り込みました。これは、単なる知識の伝達ではなく、実際のビジネスシーンでAIを使いこなし、課題解決に貢献できる人材を育成することを目的としています。
    *   特に、AIのような新しい技術は、実際に手を動かし、試行錯誤する中でしか本当に身につかないという考え太郎さんのご意見を反映しています。「専門学校ではないが、無償で参加できる社会実証企業を政府が作り、そこに短期間入社させる」といった具体的なアイデアを、政策としてより汎用的な表現に落とし込みました。

2.  **参加支援と経済的負担の軽減:**
    *   スキル習得の機会を広く提供するため、これらのプロジェクトやプログラムへの参加を無償または低負担で実現することを目指す旨を追記しました。これにより、個人の経済状況に関わらず、意欲ある人がAIスキルを習得できる環境を整備します。

3.  **中小企業・地方への貢献の明確化:**
    *   支援の成果が日本の産業全体の底上げに繋がるよう、特に中小企業や地方への就職・転職が促進されるような支援を行う方針を明確にしました。これは、育成されたAI人材が、特に人材不足が深刻な中小企業や地方の活性化に貢献することを期待するものです。「大手に入ったら意味がない」という、考え太郎さんの「支援の恩恵がどこへ届くべきか」という視点を反映しています。

**背景と意図:**

日本企業のAI導入は進んでいるものの、PoCの段階で停滞したり、効果を実感できていない企業が多いという現状認識があります。その大きな要因として、AIを実務で活用できる人材、特に中小企業や地方におけるAI推進リーダーやデータエンジニアの不足が挙げられています。

今回の修正は、この課題に対し、より実効性のある人材育成策を提示することを目的としています。単にAIの知識を持つ人材を増やすだけでなく、企業が「この人ならAI活用を任せてみよう」と思えるような、実践力と課題解決能力を兼ね備えた人材を育成し、その活躍の場を中小企業や地方にも広げていくことで、日本全体の産業競争力強化を図ることが狙いです。

考え太郎さんとの議論の中で、「大学の座学のようなものではAIスキルは身につきにくい」「実際にビジネスの現場で使う経験が必要」といったご意見をいただき、今回の修正に至りました。

この変更により、より具体的で実効性の高いAI人材育成支援策となり、日本の産業界全体のAIシフトを加速させる一助となることを期待します。

    建築業界、特に中小零細企業においては、深刻な人手不足や高齢化が進行する一方で、IT導入やDX化が他業種に比べて遅れているという現状認識があります。大手企業ではBIM（Building Information Modeling）のような高度なツールの導入が進みつつあるものの、中小零細企業にとっては資金的・技術的なハードルが高く、導入が進んでいません。現場レベルでは、依然として紙ベースの図面や手作業による情報共有が多く、非効率な業務プロセスが残存しています。この結果、業界全体として生産性の向上や働き方改革が思うように進まないという課題がありました。「みんなで一斉に取り組まなければ効果が出にくい」という状況が、DX推進の足かせになっているという現場の声も考慮しました。

*   **提案内容と目的:**
    上記課題を踏まえ、現場の実情に即したDX推進を支援するため、「１）すべての企業のAIシフトを実現します」の「政策概要」に以下の具体的な施策を追加しました。
    1.  **初期ステップとしてのデジタル化奨励:** 中小零細企業も取り組みやすい「Excel・電子帳票・スマホアプリ」等を活用した業務プロセスのデジタル化を初期ステップとして奨励します。その際、データのサイロ化を防ぎ、将来的な連携や活用を見据えて、オープン形式（CSV等）でのデータ利用を推奨します。これにより、導入のハードルを下げ、まずは「小さく始める」デジタル化を後押しします。
    2.  **段階的ステップアップ支援:** 初期デジタル化を達成した企業向けに、図面・情報の電子化・標準化、さらにはBIM導入へと段階的にステップアップできるよう技術指導やコンサルティング支援を強化します。一足飛びの高度化ではなく、着実なステップを踏むことで、現場の混乱を避けつつ、徐々にDXのレベルを向上させることを目指します。
    3.  **伴走型の支援体制:** これらの取り組みに対し、導入規模やステップに応じた補助金制度や、現場に寄り添った具体的なサポート体制を構築します。単なるツールの導入支援に留まらず、デジタル化のメリットを実感し、自律的に改善を進められるような「成長体験」を伴走支援することで、やらされ感のない、実質的なDXの浸透を図ります。
    4.  **公共事業調達基準への反映検討:** 公共事業の調達基準において、段階的なデジタル化の達成度を評価項目に加えることを検討します。これにより、業界全体のDX推進に向けたインセンティブを創出します。

*   **期待される効果:**
    これらの施策を通じて、建築業界におけるDXの裾野を広げ、中小零細企業を含む業界全体の生産性向上、働き方改革、および品質向上に貢献することを目指します。

**2. AI活用におけるデータ整備の重要性の明確化**

*   **背景・課題認識:**
    AI技術の導入・活用が多くの産業で期待される一方で、その効果を十分に引き出せていない企業が少なくありません。特に、AIが学習・分析するための「質の高いデータ」が企業内に整備されていないことが、AI導入の初期段階でつまずく大きな要因の一つとして挙げられます。「現状認識・課題分析」においても、多くの企業で既存の業務データの整理や構造化が不十分であり、データの粒度が粗く、デジタル化しづらい形式で情報が蓄積されているケースが見受けられました。

**3. 提案の背景・課題**
日本の企業の99%を占める中小企業は、日本経済の持続的な成長と競争力強化の鍵を握っています。しかしながら、多くの中小企業にとって、AIは依然として遠い存在であり、その導入や活用には多くの課題が存在します。
既存の政策案では、AI人材育成の重要性が指摘されていますが、それだけでは「AIをどのように自社のビジネスに活かせるのか」「導入コストや専門知識の不足をどう補うのか」といった中小企業経営者の具体的な疑問や不安に応えきれていない可能性がありました。

    *   **導入ハードルの高さ**: 何から手をつけて良いかわからない、専門知識を持つ人材がいない、初期投資の負担が大きい。

    *   **相談相手の不在**: AI導入に関する疑問や課題を気軽に相談できる相手が身近にいない。
    *   **試行機会の不足**: リスクを恐れて、実際にAIを試してみる機会がない。

    *   **実践的な活用促進**: アカウントを通じて、業界別のAI活用事例の共有や、簡単なAIツールの操作研修なども提供し、企業がAIを実務で活用していくための具体的な道筋を示します。

**5. 期待される効果**
本施策の導入により、以下のような効果が期待されます。
    *   **AI導入の裾野拡大**: 中小企業におけるAI導入の心理的および技術的なハードルを大幅に引き下げ、これまでAI活用に踏み出せなかった企業層への普及を促進します。
    *   **実践的なAI活用スキルの習得**: 企業が実際にAIツールを操作し、自社のデータで試行錯誤する中で、実践的なAI活用スキルやノウハウが蓄積されます。
    *   **具体的な業務改善と価値創出**: 伴走型サポートにより、AIが各企業の個別の課題解決に結びつきやすくなり、業務効率化、コスト削減、新サービス開発といった具体的な成果を生み出すことが期待できます。これにより、「PoCの墓場」に陥るリスクを低減します。

この変更は、きしもとさんのご提案に基づき、日本の産業が直面する重要な課題の一つである企業間取引のデジタル化の遅れに対応するためのものです。

### 変更の背景と目的

現状の政策案「４．産業」における「２）AI・ITの活用により便利で効率的な産業支援を実現します」では、主に政府の支援制度のデジタル化に焦点が当てられていました。しかし、きしもとさんの分析によると、日本の多くの企業、特に中小企業は、電話、FAX、紙の請求書といったアナログな商習慣に依然として大きく依存しており、これが生産性向上やデータ活用の大きな障壁となっています。

特に深刻なのは、個別の企業がデジタル化を試みても、取引先が従来のアナログな手法を継続している場合、デジタルとアナログの二重管理が発生し、かえって業務負担が増大してしまうという「デジタル化のジレンマ」です。この問題が、多くの企業がDX推進に踏み切れない大きな要因となっています。

本提案の目的は、この膠着状態を打破し、企業が真のDXの恩恵を享受できるようにすることです。具体的には、行政が主導して、受発注、請求、決済といった企業間取引プロセス全体をカバーする共通デジタルインターフェースを提供することで、個社最適ではなく、業界全体、さらには社会全体の効率化を目指します。

    *   これにより期待される多岐にわたる効果を具体的に記述：

        *   粉飾決算などの不正会計リスクの低減（取引の透明性向上）
